# 💖 Мой Путь — Трекер привычек

PWA приложение для отслеживания личных привычек с философией **самолюбви и заботы**, а не контроля и наказания.

## 🌟 Особенности

- ✨ **Мобильное веб-приложение (PWA)** — работает без интернета
- 💾 **Локальное хранение** — все данные хранятся на устройстве
- 🎨 **Мягкий дизайн** — пастельные тона (розовый, персиковый, мятный)
- 📊 **Аналитика** — тепловая карта активности и статистика
- 🔥 **Серии дней** — отслеживание последовательности выполнения
- 🎯 **9 привычек по умолчанию** — утро/день/вечер
- ➕ **Кастомные привычки** — добавляй свои практики
- 📱 **Адаптивный дизайн** — идеален для мобильных устройств
- 🎉 **Анимации** — конфетти при достижении 100% прогресса
- 💬 **Рефлексия** — утренняя и вечерняя практики
- 🧘 **Спиральная динамика** — упражнения для гармонии тела и сознания

## 🚀 Начало работы

### Требования
- Node.js 16+ 
- npm или yarn

### Установка

```bash
# Перейди в папку проекта
cd habit-tracker

# Установи зависимости
npm install

# Запусти dev сервер
npm run dev
```

Приложение откроется на `http://localhost:5173`

### Сборка для продакшена

```bash
npm run build
npm run preview
```

## 📱 Установка как приложение

После открытия приложения в браузере:

**На Android:**
1. Нажми меню браузера (три точки)
2. Выбери "Установить приложение" или "Add to Home screen"

**На iOS:**
1. Нажми кнопку "Поделиться"
2. Выбери "На экран Домой"

**На Windows/Mac:**
1. Откройте приложение в Chrome
2. Нажми иконку установки в адресной строке
3. Нажми "Установить"

## 🎯 Структура экранов

### Экран "Сегодня" ☀️
- Дата и день недели
- Приветствие с именем
- Привычки, сгруппированные по времени
- Прогресс-бар дня (0-100%)
- Анимация при выполнении всех привычек

### Экран "Прогресс" 📊
- Тепловая карта активности (как на GitHub)
- Текущая серия 🔥 дней подряд
- Статистика по каждой привычке (7/30 дней)
- Процент выполнения

### Экран "Мои привычки" ⚙️
- Список всех привычек
- Добавление новых привычек
- Редактирование названий
- Удаление кастомных
- Активация/деактивация
- Группировка по времени

### Экран "Профиль" 👤
- Имя пользователя
- Дата начала пути
- Любимая аффирмация
- Настройки уведомлений (время снос, день, вечер)
- Экспорт данных в JSON
- Очистка всех данных

## 📊 Привычки по умолчанию

### Утренние практики ☀️
1. Правильно проснуться — пробуждение с благодарностью
2. Выпить стакан воды — увлажнение организма
3. Спиральная динамика (5-7 упражнений) — гармония тела и сознания
4. Утренняя рефлексия — ответы на 3 вопроса о ценности дня

### Дневные практики 🌤️
5. Прогулка ≥ 1 часа — движение и свежий воздух
6. Выпить 1,5-2л воды — постоянное увлажнение

### Вечерние практики 🌙
7. Вечерняя рефлексия — благодарность и удачи дня
8. Отметить настроение — выбрать одну из 5 эмоций
9. Полноценный сон (8 часов) — восстановление

## 🛠️ Технологии

- **Frontend**: React 18 + JSX
- **Сборщик**: Vite
- **Стили**: Tailwind CSS 3
- **Дата**: date-fns
- **PWA**: Service Worker + Web App Manifest
- **Хранение**: localStorage
